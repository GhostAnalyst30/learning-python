# 🐍 Learning Python

Repositorio personal para aprender Python desde cero hasta nivel avanzado, organizado por módulos temáticos con ejercicios, notas y proyectos.

---

## Índice

| # | Módulo | Descripción |
|---|--------|-------------|
| 00 | **Fundamentals** | Sintaxis básica, variables, tipos de datos, operadores y estructura del lenguaje. |
| 01 | **Embedded Functions** | Funciones incorporadas, estructuras de control (`if`, `for`, `while`), comprehensions y manejo de errores. |
| 02 | **Advanced Python** | Decoradores, generadores, context managers, programación funcional y módulos avanzados. |
| 03 | **Algorithms** | Algoritmos clásicos, estructuras de datos, búsqueda, ordenamiento y complejidad computacional. |
| 04 | **Automation** | Automatización de tareas con scripts, manipulación de archivos y procesos del sistema. |
| 05 | **Data Base** | SQL, bases de datos relacionales y no relacionales, ORMs y conexión con Python. |
| 06 | **Data Science** | NumPy, Pandas, Matplotlib, análisis exploratorio y visualización de datos. |
| 07 | **Machine Learning** | Scikit-learn, regresión, clasificación, clustering y pipelines de ML. |
| 08 | **Deep Learning** | TensorFlow / PyTorch, redes neuronales, CNN, RNN y transfer learning. |
| 09 | **Artificial Vision** | OpenCV, procesamiento de imágenes, detección de objetos y visión por computadora. |
| 10 | **Modern IA** | LLMs, embeddings, RAG, agentes inteligentes y herramientas modernas de IA. |
| 11 | **Web Development** | Flask / FastAPI / Django, APIs REST, plantillas y frontend básico. |
| 12 | **DevOps** | Docker, CI/CD, testing, logging y despliegue de aplicaciones Python. |
| 13 | **CyberSecurity** | Criptografía, análisis de vulnerabilidades, hardening y seguridad en Python. |
| 14 | **Web Scraping** | BeautifulSoup, Selenium, Scrapy, extracción y automatización web. |
| 15 | **Data Engineering** | ETL, pipelines de datos, big data, Spark y procesamiento distribuido. |
| 16 | **MultiAgent Systems** | Frameworks multi-agente (CrewAI, AutoGen), orquestación y agentes autónomos. |

---

## Estructura del repositorio

```
learning-python/
├── exercises/       # Ejercicios prácticos por módulo
├── notes/           # Notas de estudio y documentación
├── projects/        # Proyectos integradores
└── README.md        # Este archivo
```

Cada módulo tiene tres carpetas: `exercises/`, `notes/` y `projects/`, numeradas del `00` al `16`.
